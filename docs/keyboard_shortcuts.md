# キーボードショートカット

Fastladder 互換のキーボードショートカット一覧。

## フィードナビゲーション（左ペイン）

| 操作 | プライマリ | 代替キー |
|------|-----------|---------|
| 次のフィード | `s` | `Shift+Ctrl` / `Shift+↓` |
| 前のフィード | `a` | `Ctrl+Shift` / `Shift+↑` |
| 左ペイン再読み込み | `r` | |
| 未読フィード先頭へ | `w` | `Shift+Home` |
| 未読フィード末尾へ | `Shift+w` | `Shift+End` |
| 左ペイン開閉 | `z` | |

## エントリナビゲーション

| 操作 | プライマリ | 代替キー |
|------|-----------|---------|
| 次のエントリ | `j` | `Enter` |
| 前のエントリ | `k` | `Shift+Enter` |
| スクロールダウン | `Space` | `PageDown` |
| スクロールアップ | `Shift+Space` | `PageUp` |
| エントリ開閉 | `c` | |
| 古いエントリを読み込み | `>` | `Shift+j` |
| 新しいエントリを読み込み | `<` | `Shift+k` |

## アクション

| 操作 | プライマリ | 代替キー |
|------|-----------|---------|
| ピン追加/解除 | `p` | |
| ピン留めエントリを新タブで開く（最大5個、開いたら解除） | `o` | |
| 現在のフィードを全既読 | `Shift+A` | |
| エントリを新タブで開く | `v` | `Ctrl+Enter` |
| 購読解除 | `Delete` | |
| 検索ボックスにフォーカス | `f` | |

## 実装メモ

- 日本語入力中 (IME composing) はショートカットを無効化する
- input/textarea にフォーカスがある場合はショートカットを無効化する（`f` で検索ボックスにフォーカスした後など）
- Stimulus の `keyboard_controller` でグローバルに `keydown` をリッスンし、各コントローラーにディスパッチする

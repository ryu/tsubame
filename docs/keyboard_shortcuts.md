# キーボードショートカット

デスクトップのみ（モバイルでは無効）。日本語入力中・input/textarea フォーカス時も無効。

## フィードナビゲーション

| キー | 操作 |
|------|------|
| `s` | 次のフィード |
| `a` | 前のフィード |
| `r` | ページ再読み込み |

## エントリナビゲーション

| キー | 操作 |
|------|------|
| `j` | 次のエントリ |
| `k` | 前のエントリ |
| `Space` | エントリ詳細をスクロールダウン |
| `Shift+Space` | エントリ詳細をスクロールアップ |

## アクション

| キー | 操作 |
|------|------|
| `v` | エントリを新タブで開く |
| `p` | ピン追加/解除 |
| `o` | ピン留めエントリを新タブで開く（開いたら解除） |
| `Shift+A` | 現在のフィードを全既読 |
| `b` | はてなブックマークページを開く |
| `h` | はてなブックマークに追加 |

## 実装

`keyboard_controller.js` でグローバルに `keydown` をリッスン。
IME composing 中 (`event.isComposing`) はスキップ。
